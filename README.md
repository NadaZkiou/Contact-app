# Contact Manager App

A simple Contact Manager app built with **React**. It allows users to add, view, edit, delete, and search contacts through a clean and responsive interface.

---

##  Features

- Add new contacts
- View contact list
- Search contacts by name
- View contact details
- Edit contact info
- Delete contacts
